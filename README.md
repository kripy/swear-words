# Yet Another List of Bad Words #

In the spirit of LDNOOBW's [List-of-Dirty-Naughty-Obscene-and-Otherwise-Bad-Words](https://github.com/LDNOOBW/List-of-Dirty-Naughty-Obscene-and-Otherwise-Bad-Words), this repo contains their base list, updated for 2018 and beyond.

The list is useful both for web form filtering, as well as exclusion lists on social media.
